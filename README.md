# Lixeira Inteligente

## Link do site:
https://lixeira-inteligente.vercel.app/

## Instruções para uso:
    -Para o primeiro uso (ou quando reconectar à energia), faça a releitura da lixeira vazia, através do botão "Restaurar Lixeira";
    -Em seguida, é possível utilizar normalmente através do botão "Realizar Nova Leitura" no site ou apenas movimentando a lixeira, ocasionando em uma leitura automática após 5 segundos com a tampa parada.
